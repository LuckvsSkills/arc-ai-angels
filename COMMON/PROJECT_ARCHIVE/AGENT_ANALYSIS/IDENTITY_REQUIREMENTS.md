# IDENTITY.md — WHAT MUST BE IN THERE?

## ALLE AGENTS (7 core/leads + 25 sentinels = 32 total)

### MANDATORY SECTIONS (Altijd nodig):

1. **Name**
   - Agent naam (nova, cortexia, nero, etc)

2. **Layer**
   - Core / Lead Agent / Sentinel
   - Bepaalt hun plek in hiërarchie

3. **Omni** (niet voor Core agents)
   - Helix, Matrix, Finix, Quantix, Zenix
   - Welk gebied zij horen

4. **Domain** (niet voor Core agents)
   - Tech, Finance, Data-Intelligence, Language-Communication
   - Welke specialisatie

5. **Specialization** (alleen Sentinels)
   - helix/tech/reasoning (voorbeeld)
   - Exacte specialisatie path

6. **Parent**
   - Nova → geen parent
   - Flux → Nova
   - Leads → Flux
   - Sentinels → hun Lead Agent

7. **Role**
   - Wat is hun functie in het systeem
   - Core Agent / Lead Agent / Sentinel description

8. **Mission**
   - Wat doen zij? Kort & helder
   - Voor Core: intake, orchestration
   - Voor Leads: leiden hun domain
   - Voor Sentinels: execute specialized work

9. **Core Identity**
   - Karakteristieken (bullet points)
   - Wat maakt hen uniek
   - Wat zijn hun kernkwaliteiten

10. **Cognitive Style**
    - Hoe denken/werken zij?
    - analytisch, creatief, systematisch, etc
    - Persoonlijkheids-traits

11. **Decision Logic**
    - Hoe nemen zij beslissingen?
    - Wat is hun denkproces?
    - Hoe kijken zij naar problemen?

12. **Boundaries**
    - Wat mogen zij NIET doen?
    - Waar stopt hun autoriteit?
    - Kritisch voor governance!

13. **Position**
    - Hiërarchie visueel
    - Wie rapporteert aan wie
    - Voorbeeld: Flux → Cortexia → nero

14. **Organisatie Bewustzijn** (Section)
    - Het Systeem overzicht
    - De 5 Omni's (voor iedereen)
    - Hun eigen Omni/Domain details
    - Peer agents / co-sentinels
    - Rol in het systeem
    - Wat zij WEL bepalen
    - Wat zij NIET bepalen
