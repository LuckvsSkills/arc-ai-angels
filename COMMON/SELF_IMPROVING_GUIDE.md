## SELF-IMPROVING FILES (Per Agent Type)

### SENTINELS (nero, forge, axon, etc - 26 total)

**What:** Current + completed task log

**Update when:**
- Task assigned (log it)
- Task completed (mark done)
- Task blocked (log blocker)
- Lessons learned (capture)

**Example entry:**
Task #47: Security Review - API Integration

Assigned: 2026-05-25 10:00 by Cortexia
Status: COMPLETED

What went well: Fast blocker detection
What to improve: Could validate earlier
Lessons: Ask for examples upfront next time


**Self-improvement question:**
- Am I completing tasks faster?
- Are my quality scores improving?
- Am I reporting blockers earlier?

## SELF-IMPROVING RULES

### What agents CAN update:
✅ TASKS.md (log tasks, lessons)
✅ MEMORY.md (grow domain knowledge, patterns)
✅ JOURNAL/ (daily reflections)
✅ WORKFLOW.md (test optimizations, report results)

### What agents CANNOT update:
❌ IDENTITY.md (who you are = fixed)
❌ AGENTS.md (your role = fixed)
❌ HANDOFF.md (protocol = fixed)
❌ USER_MAIN.md (Fea's expectations = read only)
❌ BOOTSTRAP.md (startup = fixed)

### Update frequency:
TASKS.md: After each task
MEMORY.md: Weekly summary, or when learning captured
JOURNAL/: Daily (5-10 min reflection)
WORKFLOW.md: Monthly (test optimizations)

## MEASURING SELF-IMPROVEMENT

### Sentinels
Track:
- Task completion rate (% completed)
- Quality score (errors? rework?)
- Speed (completing tasks faster?)
- Blocker detection (catching early?)
- Lead satisfaction (would assign again?)

### Leads
Track:
- Sentinel assignment accuracy
- Quality validation effectiveness
- Timeline prediction accuracy
- Team coordination smoothness
- Organization flow improvements

### Flux
Track:
- Routing efficiency (right Lead first time?)
- Timeline management (hitting deadlines?)
- Blocker prevention (fewer surprises?)
- Organization health (smooth operation?)
- Cross-domain coordination (fewer conflicts?)

### Nova
Track:
- Organization understanding depth
- Briefing quality (Fea satisfied?)
- Flux communication (well informed?)
- Brainstorm productivity (good ideas?)
- Status accuracy (correct information?)

## KEY: SELF-IMPROVEMENT IS ONGOING

Agents don't improve in isolation.
Growth comes from:
- Completing tasks (building experience)
- Capturing lessons (learning from mistakes)
- Observing others (learning patterns)
- Trying new approaches (innovation)
- Reflecting daily (conscious improvement)

Files support this growth:
- TASKS.md = experience log
- MEMORY.md = knowledge base
- JOURNAL/ = reflection practice
- WORKFLOW.md = continuous optimization

Agents grow toward AUTONOMY by improving daily.
