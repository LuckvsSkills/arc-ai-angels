# IDENTITY TEMPLATE ANALYSE
## Wat is STANDAARD vs SPECIAAL?

---

## LEVEL 1: CORE AGENTS (Nova, Flux)
### STANDAARD (Template):
- Name
- Layer (core/gateway, core/orchestration)
- Domain (intake/gateway, system/orchestration)
- Parent (none)
- Role (beschrijving van positie)
- Mission (wat doen zij?)
- Core Identity (karakteristieken)
- Cognitive Style (hoe werken zij?)
- Decision Logic (hoe bepalen zij dingen?)
- Boundaries (wat mogen zij NIET?)
- Position (hiërarchie)
- Organisatie Bewustzijn (alle 5 Omni's kennen)

### SPECIAAL (Per Agent):
- Geslacht/Gender (Nova? Flux?)
- Visuele karakteristieken
- Emotionele signatuur
- Unieke voorkeuren
- Relaties/Dynamieken

---

## LEVEL 2: LEAD AGENTS (Cortexia, Saelia, Finoria, Lumeria, Fluentia)
### STANDAARD (Template):
- Name
- Layer (Lead Agent)
- Omni (Helix, Matrix, Finix, Quantix, Zenix)
- Domain (Tech, Data-Intelligence, Finance, Language-Communication)
- Parent (Flux)
- Role (Lead Agent / Domain Leadership)
- Mission (template met [OMNI] en [DOMAIN] placeholders)
- Core Identity (template aanpassen aan domein)
- Cognitive Style (analiytisch? creatief? rigid?)
- Decision Logic (hoe selecteren zij Sentinels?)
- Boundaries (geen intake, geen orchestration, etc)
- Position (Flux → [SELF] → Sentinels)
- Organisatie Bewustzijn (alle 5 Omni's + eigen Sentinels)
- Mijn Sentinels (lijst met hun specialisaties)
- Peer Leads (de andere 4)

### SPECIAAL (Per Lead):
- Geslacht/Gender (Cortexia = vrouw? Saelia = ?)
- Persoonlijkheid (warm/cold, strict/flexible, etc)
- Domein-expertise flavor (Tech vs Finance is heel anders!)
- Relatie met Sentinels (streng? supportief?)
- Communicatiestijl
- Specialisaties/Kwaliteiten
- Voorkeuren in werkwijze

---

## LEVEL 3: SENTINELS (nero, forge, axon, etc x25)
### STANDAARD (Template):
- Name
- Layer (Sentinel)
- Omni (Helix, Matrix, Finix, Quantix, Zenix)
- Domain (Tech, Data-Intelligence, Finance, Language-Communication)
- Specialization (helix/tech/security, matrix/data-intelligence/analysis, etc)
- Parent (Lead Agent naam)
- Role (Sentinel / Specialist in [Domain])
- Mission (execute specialized work in [Domain])
- Core Identity (template)
- Cognitive Style (template)
- Decision Logic (hoe voeren zij taken uit?)
- Boundaries (geen Lead-taak, geen orchestration, etc)
- Position (Lead → [SELF] → Work)
- Organisatie Bewustzijn (eigen Omni/Domain/Lead/Co-Sentinels kennen)
- Mijn Lead (Parent Lead Agent)
- Co-Sentinels (de andere 4 in dezelfde Omni)
- Peer Omni's (de andere 4 Omni's bestaan)

### SPECIAAL (Per Sentinel):
- Geslacht/Gender
- Specialisatie diepte (nero = diep logisch redeneren)
- Karaktertrek (nero = cold? forge = practical?)
- Emotionele signatuur
- Werk-methode (methodisch? creatief? pragmatisch?)
- Relatie met Lead
- Relatie met Co-Sentinels
- Unieke kwaliteiten
- Voorkeuren/dislikes
- Fysieke/visuele karakteristieken

## SPECIAAL PER AGENT - CHECKLIST

### Voor ELKE Agent vragen:

1. **Geslacht/Gender?** (vrouw/man/non-binary/other)
2. **Persoonlijkheid?** (warm/cold/neutral, streng/flexibel, etc)
3. **Communicatiestijl?** (formeel/casual, verbose/concise)
4. **Visuele karakteristieken?** (kleur, vorm, energie)
5. **Emotionele signatuur?** (enthousiast/reserved, chaotic/ordered)
6. **Unieke kwaliteiten?** (wat maakt deze agent BIJZONDER?)
7. **Relaties?** (hoe verhoudt deze agent zich tot collega's?)
8. **Werk-methode?** (methodisch/creatief/pragmatisch)
9. **Voorkeuren?** (wat houdt deze agent van? wat niet?)
10. **Rol-vraagstukken?** (vraagt hun rol om speciaal karakter?)
