## 2. OVERDRACHT ROUTES

### Route 1 — Code Only (Basis)
**Voor:** Technische klanten met eigen development team
**Prijs:** Gratis bij projectprijs boven €3.000 / €250 bij kleinere projecten
**Wat wordt geleverd:**
- Volledige broncode via GitHub repository (private of public)
- README.md met installatie instructies
- .env.example met alle environment variables
- Database schema en migrations
- Deploy instructies voor Vercel/VPS

**Verantwoordelijkheid klant:** Alle technische zaken na overdracht

---

### Route 2 — Code + Documentatie (Standaard)
**Voor:** Klanten met enige technische kennis
**Prijs:** €500 toeslag
**Wat wordt geleverd:**
- Alles van Route 1
- HANDOFF.md — gebruikershandleiding admin panel
- SOP.md — dagelijkse en wekelijkse procedures
- DEPLOYMENT.md — stap voor stap deploy handleiding
- Video walkthrough (30 min opname van admin panel)
- 14 dagen email support na overdracht

**Verantwoordelijkheid klant:** Technische hosting en updates

---

### Route 3 — Code + Training (Premium)
**Voor:** Klanten zonder technische kennis die zelf willen beheren
**Prijs:** €1.000 toeslag
**Wat wordt geleverd:**
- Alles van Route 2
- 2 uur live training sessie (video call)
- Training recording beschikbaar na sessie
- Aangepaste handleiding op maat
- 30 dagen support via Telegram (werkdagen)
- Eenmalige gratis wijziging na training

**Verantwoordelijkheid klant:** Dagelijks beheer na training

---

### Route 4 — Volledige Overdracht + SLA (Enterprise)
**Voor:** Bedrijven die complete overdracht willen maar nog ondersteuning nodig hebben
**Prijs:** €2.500 toeslag + maandelijks SLA pakket
**Wat wordt geleverd:**
- Alles van Route 3
- Volledige IP overdracht en documentatie
- Technische overdracht meeting met dev team klant
- 90 dagen support en begeleiding
- SLA naar keuze (zie sectie 5)
- Optie om later terug te schakelen naar ARC beheer

### Per template bevat
/frontend          — HTML/React code

/backend           — FastAPI (indien van toepassing)

/admin             — Admin panel (indien van toepassing)

/database          — Schema + migrations

/agent             — OpenClaw agent config (indien tier 3+)

/deploy            — Vercel config + scripts

/docs              — README, HANDOFF, SOP

.env.example       — Environment variables

vercel.json        — Deploy configuratie

package.json       — Dependencies

### Stijl injectie via CSS variabelen

```css
:root {
  --bg:        [kleur];

  --textMuted: [kleur];
  --border:    [kleur];
  --font-heading: '[font]', sans-serif;

}
```
