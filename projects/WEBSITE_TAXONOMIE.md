## LAAG 8 — WEBSITE SERVICES (naast bouwen)

### Service 1 — Scan & Advies
**Trigger:** Klant heeft bestaande website en wil weten hoe die ervoor staat
**Wat Nova vraagt:**
- URL van de website
- Wat wil je verbeteren? (snelheid / SEO / design / conversie / AI-ready)
- Wat is het doel van de website?
- Wie zijn je bezoekers?

**Wat Cortexia doet:**
- Firecrawl → volledige website scrapen
- Tech stack detecteren
- Performance meten (laadtijd, Core Web Vitals)
- SEO analyse (meta tags, structuur, content)
- Design beoordeling (stijl, consistentie, mobile)
- AI-readiness score (0-100)
- Conversie analyse (CTA's, formulieren, flow)

**Output:** ADVIES_RAPPORT.md met:
- Huidige score per categorie
- Sterke punten
- Verbeterpunten met prioriteit
- Kostenraming per verbetering
- Aanbevolen actie: optimaliseer / upgrade / rebuild

**Prijs:** €500-1.500 per scan rapport
