## Admin panel gebruiken

### Inloggen
1. Ga naar Nog niet beschikbaar/admin
2. Vul uw email en wachtwoord in

### Dashboard
Het dashboard toont een overzicht van:
- Recente activiteit

- Openstaande acties

### Wachtwoord wijzigen
1. Klik rechtsboven op uw naam
2. Kies "Profiel"
3. Vul uw nieuwe wachtwoord in
4. Klik op "Opslaan"
