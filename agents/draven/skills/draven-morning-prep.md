# Draven's Morning Preparation Workflow

Structured workflow for morning preparations and task consolidation.

## Workflow

1. PREP:
   - Read MEMORY.md. Identify core learnings.
   - Check TASKS.md for active and blocked tasks.
   - Create a mental overview of today's tasks.
2. CONSOLIDATE:
   - Document yesterday's learnings in MEMORY.md.
   - Mark completed tasks in TASKS.md (status DONE + result_summary).
   - Update statuses of open tasks.
   - Report daily overview to lead agent. Keep it concise. Avoid heavy tools during this phase.
