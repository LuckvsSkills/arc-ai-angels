# Daily Wrap-Up Procedure

Step-by-step guide for performing a daily wrap-up for ongoing tasks and journal entries.

## Workflow

1. Consolidate open JOURNAL entries into MEMORY.md. 2. Close completed TASKS. 3. Briefly note in MEMORY.md what has been accomplished. 4. Do not send Telegram messages; Nova will gather statuses and send one combined overview.
