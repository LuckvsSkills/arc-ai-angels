# Daily Wrap-Up Workflow

Systematic procedure for completing daily wrap-up tasks including task updates and insights in the memory, with updates based on recent experiences.

## Workflow

1. **Dag Afsluiting:**  
   - Update TASKS.md with completed (DONE), in-progress, and blocked tasks.  
   - Note any blocked reasons if applicable.  
2. **Memory Update:**  
   - Update MEMORY.md with key learnings and insights.  
   - Remove outdated information.  
3. **Rapportage:**  
   - Report daily results concisely to the Omni Lead.
