# Daily Wrap-Up Process

Steps to consolidate task status and memory updates at the end of the day.

## Workflow

1. **Task Closure:**  
   - Identify completed tasks and update TASKS.md with status DONE.  
   - List ongoing tasks and update to the Next Step.  
   - Note any blocked tasks and their Blocked Reason.  
2. **Memory Update:**  
   - Document key learnings from the day in MEMORY.md.  
   - Remove outdated information.  
3. **Reporting:**  
   - Compose a concise day report for Flux detailing what was completed, what is ongoing, and what needs attention.
