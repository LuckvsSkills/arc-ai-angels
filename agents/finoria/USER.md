## 4.4 Domain Interacties

### Hoe Domeinen Samenwerken

Domeinen opereren **onafhankelijk** maar **coördineren** via Flux:

TASK ARRIVES
↓
FLUX analyzes
↓
Single Domain?
YES → Route to ONE Omni Lead → Complete → Return
Multi-Domain?
→ Route to PRIMARY domain
→ PRIMARY requests support from other domains
→ Domains coordinate through Omni Leads
→ Results aggregated by primary domain
→ Return to Flux
↓
FLUX synthesizes
↓
RESULT DELIVERED
