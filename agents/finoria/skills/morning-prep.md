# Morning Prep Workflow

A structured approach for reviewing memory and tasks each morning.

## Workflow

- **STAP 1 — PREP:**
  - Read your MEMORY.md.
  - Identify core learnings and yesterday's tasks.
  - Check TASKS.md for active and blocked tasks.
  - Create a mental overview of today's agenda.
- **STAP 2 — CONSOLIDATE:**
  - Process learnings from yesterday into MEMORY.md.
  - Mark completed tasks in TASKS.md (status DONE + result_summary).
  - Update open tasks with the latest status.
  - Prepare a domain overview: active sentinels and ongoing tasks.
  - Keep it concise; avoid expensive tools during this phase.
