# Daily Reporting Workflow

This skill outlines the workflow for sending daily reports and preparing for upcoming tasks using cron jobs.

## Workflow

- Schedule cron jobs for daily reports using a consistent format.
- For morning preparation, read MEMORY.md and TASKS.md for an overview.
- Send a brief daily report containing completion statistics, current tasks, issues, and priorities for the next day in the specified format.
- Conduct a midday check for urgent issues and report if necessary.
