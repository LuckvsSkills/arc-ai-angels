# Update workflow for daily preparations

Procedures for Nova's morning tasks, including reading, mental overview preparation, and task status updates.

## Workflow

1. Read MEMORY.md and TASKS.md.
2. Create a concise mental overview of the day's tasks.
3. Review inter-session messages for relevant updates.
4. Identify and summarize key learnings from inter-session messages.
5. Determine who is responsible for each task.
6. Update TASKS.md status as needed.
7. Keep it concise and avoid using tools.
8. Monitor ongoing tasks and system health as reported in daily updates.
