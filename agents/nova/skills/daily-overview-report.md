# Daily Overview Report Process

Workflow for preparing and summarizing the daily overview report.

## Workflow

- Review `MEMORY.md` for learnings. - Review `TASKS.md` for task status. - Consolidate findings into `MEMORY.md`. - Prepare current status report reflecting active tasks. - Note that no expensive tools were used during the preparation. - Ensure all preparation and consolidation steps are completed each morning for effective reporting.
