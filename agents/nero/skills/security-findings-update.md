# Security Findings Update

Capture the process of reviewing daily security findings and updating security memory with insights.

## Workflow

1. Conduct a search for daily security findings via a trusted source.
2. Analyze the results for critical vulnerabilities and technologies at risk.
3. Document highlights in MEMORY.md for ongoing reference.
4. Identify any necessary updates to security policies based on findings.
