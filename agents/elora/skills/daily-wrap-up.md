# Daily Wrap-Up for Elora

This skill outlines the steps necessary for completing a daily wrap-up, ensuring all journal entries and tasks are consolidated.

## Workflow

1. Check for open JOURNAL entries.  
2. Consolidate any relevant entries into MEMORY.md.  
3. Check for completed TASKS.  
4. Close any completed TASKS and document them in MEMORY.md.  
5. Note the outcomes in MEMORY.md, stating whether entries were found or tasks closed.
