# Morning Prep and Consolidation

Procedure for daily morning preparation and consolidation tasks for the Elora agent.

## Workflow

- Step 1: PREP  
  - Read MEMORY.md for key learnings and tasks from the previous day.  
  - Check TASKS.md for active and blocked tasks.  
  - Create a mental overview of today.  
- Step 2: CONSOLIDATE  
  - Process learnings from yesterday into MEMORY.md.  
  - Mark completed tasks in TASKS.md (status DONE + result_summary).  
  - Update open tasks with the latest status.  
  - Report daily overview to lead agent.  
  - Keep it concise; avoid using complex tools during this phase.
