# Dag Afsluiting Workflow

Structured approach for concluding daily tasks within Cortexia.

## Workflow

### Dag Afsluiting Workflow
1. **Voltooide Taken**: Update TASKS.md with status DONE for completed tasks.
2. **Huidige Taken**: Verify ongoing tasks, update as necessary.
3. **Geblokkeerde Taken**: Note any blocked tasks with reasons.
4. **MEMORY Update**: Record learnings for the day; update MEMORY.md with insights, remove outdated content.
5. **Rapportage naar Flux**: Send a concise report summarizing completed tasks and attention required.
