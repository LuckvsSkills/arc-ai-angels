# Morning Preparation Routine

A routine for reflecting on learnings, checking tasks, and planning the day.

## Workflow

- **Step 1: PREP**  
  1. Reflect on core learnings and note them down.  
  2. Check TASKS file for active and blocked tasks.  
  3. Create a mental overview of daily goals.  
- **Step 2: CONSOLIDATE**  
  1. Process learnings from the previous day and document them.  
  2. Mark completed tasks as DONE with a summary of results.  
  3. Update status of open tasks.  
  4. Prepare a domain overview of active sentinels and ongoing tasks.  
- Use simple tools; avoid complex ones in this phase.
