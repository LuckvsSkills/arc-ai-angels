# Daily Morning Preparation Workflow

Procedure for effective daily preparation and task consolidation.

## Workflow

1. PREP: - Read MEMORY.md for key learnings. - Review TASKS.md for active and blocked tasks. - Create a mental overview of the day. 2. CONSOLIDATE: - Process learnings from yesterday into MEMORY.md. - Complete finished tasks in TASKS.md (set status to DONE and include result summary). - Update ongoing tasks with the latest status. - Prepare the domain overview (active sentinels and ongoing tasks). - Keep it concise. - Avoid expensive tools (firecrawl/exa/perplexity) during this phase.
