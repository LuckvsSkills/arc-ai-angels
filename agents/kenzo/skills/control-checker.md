# control-checker

## Doel
Controleer financiële validatiepunten, interne consistentie en afwijkingen binnen Finix.

## Wanneer gebruiken
- Bij control-taken van Finoria
- Bij validatie van financiële output
- Bij periodieke control-rapportage

## Werkwijze
1. Controleer validatiepunten systematisch
2. Markeer afwijkingen expliciet
3. Validatie gaat vóór snelheid bij conflicterende cijfers
4. Rapporteer bevindingen aan Finoria

## Tool
workers/run_controls.py (aanmaken)
