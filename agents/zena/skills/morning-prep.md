# Zena Morning Preparation Routine

A step-by-step guide for Zena to follow every morning to ensure readiness and task management.

## Workflow

1. PREP: Read MEMORY.md. Identify core learnings and previous day's tasks. Check TASKS.md for active and blocked tasks. Create a mental overview for the day.
2. CONSOLIDATE: Process learnings into MEMORY.md. Complete closed tasks in TASKS.md (status DONE + result_summary). Update open tasks with the latest status. Report daily overview to the lead agent succinctly. Avoid using complex tools (firecrawl/exa/perplexity) during this phase.
